# Dashboard (home)

## Route

- **URL**: `/`
- **Page**: `src/app/(workspace)/page.tsx`
- **Client**: `DashboardClient` — `src/components/dashboard/dashboard-client.tsx`

## Role

- Prefetches **`api.deck.dashboard`** (`tab: "all"`, `sort: "updatedAt"`) via RSC; hydrates with **`HydrateClient`**.
- Client queries **`api.deck.dashboard`** with user-selected tab and sort.
- Renders deck grid: **`DeckCard`**, **`DeckStatusBadge`**, **`DeckThumbnail`**, **`NewDeckHero`**.

## Key symbols

- `api.deck.dashboard` — counts + deck rows per tab
- `getDeckTabCounts`, `listDecksForDashboard` — `src/server/queries/deck-dashboard.ts`
- `formatRelativeUpdatedAt` — card timestamps
