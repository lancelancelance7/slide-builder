# New deck

## Route

- **URL**: `/decks/new` (optional `?mode=template` → `templateMode` on client)
- **Page**: `src/app/(workspace)/decks/new/page.tsx`
- **Client**: `NewDeckPageClient` — `src/components/decks/new/new-deck-page-client.tsx`

## Role

- **Server**: Loads brand kits from Drizzle (not tRPC). If none exist, shows CTA to `/brand-kits/new`.
- **Client**: Brief (`NewDeckBriefForm`), constraints (`NewDeckConstraintsRow`), plan settings (`NewDeckPlanSettings`), starters (`NewDeckSampleStarters`), stepper (`NewDeckStepper`), **`OpenAiModelSelect`**.
- Submits **`api.deck.create`** with `settings` (slide count range, audience, tone, layouts, image/notes policies, `requirePlanReview`) and `afterCreate: "plan"` → navigates to `/decks/[id]/plan`.

## Key symbols

- `api.deck.create` — `resolveBrandKitForNewDeck` on server
- `DEFAULT_NEW_DECK_LAYOUTS`, `NewDeckPlanSettings` — allowed layout pickers
- `readStoredOpenAiChatModel` — model persisted for plan page (plan itself runs on plan route)
