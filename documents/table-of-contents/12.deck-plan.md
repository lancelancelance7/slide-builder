# Deck plan

## Route

- **URL**: `/decks/[id]/plan`
- **Page**: `src/app/(workspace)/decks/[id]/plan/page.tsx`
- **Client**: `DeckPlanPageClient` — `src/components/decks/plan/deck-plan-page-client.tsx`

## Role

- **`api.slide.planBundle`** — deck, brand kit, slides list.
- On mount: if status is `draft`/`planned` and **no slides**, auto-runs **`api.ai.planDeck`** once (uses stored OpenAI model).
- **`api.ai.regeneratePlan`** — full replan.
- Per-slide: **`api.slide.update`**, **`api.ai.rewriteSlide`**, reorder/add/remove via **`api.slide.*`**.
- **`api.ai.generateSlides`** — marks `generated`, invalidates dashboard, **`router.push`** to `/decks/[id]/edit`.

## Primary UI

- `PlanPageHeader` — title, regen, generate, model select
- `PlanStatsStrip` — slide count / layout summary
- `PlanSlideCard` — inline plan fields per layout
- `NewDeckStepper` — step indicator (shared with new deck)
- `OpenAiModelSelect` — `src/components/ai/open-ai-model-select.tsx`

## Key symbols

- `generateDeckPlanViaOpenAi`, `rewriteSlideViaOpenAi` — `src/server/ai/deck-plan-openai.ts`
- `contentToFormFields`, `formFieldsToContent` — plan card editing
