# Deck editor

## Route

- **URL**: `/decks/[id]/edit`
- **Page**: `src/app/(workspace)/decks/[id]/edit/page.tsx`
- **Client**: `DeckEditorPageClient` — `src/components/decks/edit/deck-editor-page-client.tsx`

## Role

- **Server gate**: Redirects to `/decks/[id]/plan` if status is `draft` or `planned`.
- **Fullscreen shell** — workspace sidebar/nav hidden (`WorkspaceShell`).
- **`api.slide.planBundle`** — same bundle as plan; canvas editing.
- Mutations: **`api.slide.update`**, **`reorder`**, **`add`**, **`remove`** (no AI rewrite on this route).

## Primary UI

- `EditorToolbar` — actions, link to print
- `EditorThumbnailRail` — slide list / selection
- `EditorInspector` — region properties
- `SlideCanvasView`, `SlideLayoutInner`, `SlideScaledFrame` — `src/components/slides/`
