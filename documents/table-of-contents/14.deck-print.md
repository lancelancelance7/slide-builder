# Deck print

## Route

- **URL**: `/decks/[id]/print`
- **Page**: `src/app/(workspace)/decks/[id]/print/page.tsx`
- **Client**: `DeckPrintPageClient` — `src/components/decks/print/deck-print-page-client.tsx`

## Role

- **Server gate**: Same as edit — redirects to plan if `draft` or `planned`.
- **Fullscreen** layout wrapper on page + shell hides chrome.
- **`api.slide.planBundle`** — read-only slide stack for print layout.
- **`DeckPrintSlides`** — renders printable slide sequence.
