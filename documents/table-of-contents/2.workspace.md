# Workspace layout

## Route

- **Layout**: `src/app/(workspace)/layout.tsx` — wraps all workspace pages.
- **Shell**: `WorkspaceShell` in `src/components/workspace/workspace-shell.tsx`.

## Role

- Loads **`getDeckTabCounts`** and a minimal brand-kit list (id, name, accent) for the sidebar.
- Renders **`WorkspaceAppNav`** and **`WorkspaceSidebar`** unless the path is a fullscreen deck route (`/decks/[id]/edit`, `/template`, or `/print`).
- Passes `counts` and `brandKitsList` into the shell; children are route pages.

## Key symbols

- `getDeckTabCounts` — `src/server/queries/deck-dashboard.ts`
- `WorkspaceSidebar`, `WorkspaceAppNav` — `src/components/workspace/`
- `isDeckFullscreenEditorPath` — hides chrome on edit/print flows
