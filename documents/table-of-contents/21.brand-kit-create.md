# Brand kit create

## Route

- **URL**: `/brand-kits/new`
- **Page**: `src/app/(workspace)/brand-kits/new/page.tsx`
- **Client**: `BrandKitCreateRedirect` — `src/components/brand-kit/brand-kit-create-redirect.tsx`

## Role

- On mount, **`api.brandKit.create`** then **`router.replace`** to `/brand-kits/[id]`.
- No form on this route — instant create with default name/colors/fonts.
