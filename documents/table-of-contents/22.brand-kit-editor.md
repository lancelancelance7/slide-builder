# Brand kit editor

## Route

- **URL**: `/brand-kits/[kitId]`
- **Page**: `src/app/(workspace)/brand-kits/[kitId]/page.tsx`
- **Client**: `BrandKitEditorClient` — `src/components/brand-kit/brand-kit-editor-client.tsx`

## Role

- Server: **`getBrandKitDetail`** — 404 via `notFound()` if missing.
- Prefetch **`api.brandKit.byId`**; client loads detail and sections:
  - `BrandKitPageHeader`, `BrandKitLogoSection`, `BrandKitPaletteSection`, `BrandKitTypographySection`, `BrandKitToneSection`, `BrandKitImageStyleSection`
  - `BrandKitLivePreview`, `BrandKitComplianceCard` (uses **`contrastRatio`**), `BrandKitAppliesToCard`
- Mutations: **`update`**, **`duplicate`**, **`delete`**, **`setDefault`**.

## Key symbols

- `getBrandKitDetail`, `DEFAULT_BRAND_COLORS`, `DEFAULT_FONT_STACK`, `DEFAULT_IMAGE_STYLE`
- `api.brandKit.*` — see [3.api-trpc.md](./3.api-trpc.md)
