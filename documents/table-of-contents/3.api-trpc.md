# tRPC API

## Route

- **HTTP**: `GET` / `POST` → `src/app/api/trpc/[trpc]/route.ts` (`/api/trpc`)
- **Router root**: `src/server/api/root.ts` → `appRouter`
- **Uploads**: `GET` / `POST` → `src/app/api/uploadthing/route.ts` (`/api/uploadthing`) — `uploadRouter` in `src/server/uploadthing/core.ts` (`slideImage`, `brandLogo`); not tRPC

## Role

| Namespace | Procedures | Used by |
|-----------|------------|---------|
| `deck` | `create`, `dashboard` | New deck, dashboard |
| `slide` | `planBundle`, `update`, `reorder`, `add`, `remove` | Plan, edit, print |
| `brandKit` | `list`, `byId`, `create`, `update`, `duplicate`, `delete`, `setDefault` | Brand kit pages |
| `ai` | `planDeck`, `regeneratePlan`, `rewriteSlide`, `generateSlideImage`, `generateSlides` | Deck plan, edit |
| `post` | `hello` | T3 scaffold only (unused in product UI) |

### `deck`

- **`create`** — Inserts deck; resolves brand kit via `resolveBrandKitForNewDeck`. `afterCreate: "plan"` → status `planned`, else `draft`.
- **`dashboard`** — `getDeckTabCounts` + `listDecksForDashboard` (tabs: `all` \| `aiPlans` \| `generated` \| `exported`; sort: `updatedAt` \| `createdAt` \| `title`).

### `slide`

- **`planBundle`** — Deck + brand kit + ordered slides for plan/edit/print.
- **`update`** — Content/layout/image/notes; may bump deck `generated` → `edited`.
- **`reorder`**, **`add`**, **`remove`** — Slide list mutations on plan/edit.

### `brandKit`

- CRUD + **`setDefault`** (transaction clears then sets one default).
- **`delete`** blocked when decks still reference the kit.

### `ai`

- **`planDeck`** / **`regeneratePlan`** — `generateDeckPlanViaOpenAi` → replace all slides → status `planned`.
- **`rewriteSlide`** — `rewriteSlideViaOpenAi` for one slide.
- **`generateSlideImage`** — `generateAndPersistSlideImage` (OpenAI Images API → UploadThing); used by deck edit image brief.
- **`generateSlides`** — Requires ≥1 slide; sets deck status `generated`.

## Server callers

- `createCaller` / `createCallerFactory` — `src/server/api/root.ts` (server-side tests or scripts).
