# Domain: slide plan & AI

Shared types and OpenAI integration used by tRPC `slide` / `ai` routers and plan UI.

## Role

### `src/lib/slide-plan.ts`

- **`SLIDE_LAYOUT_VALUES`** / **`SlideLayoutId`** — `title`, `section`, `imageText`, `quote`, `comparison`, `statHero`, `closing`
- **`slideLayoutSchema`**, **`aiPlanSlideRowSchema`**, **`aiPlanDeckResultSchema`**
- **`defaultContentForLayout`**, **`normalizeContentForPersist`**, **`aiRowToStoredContent`**
- **`imageUrl`**, **`imageAsset`** (`slideImageAssetSchema`, `readSlideImageFromContent`) — hero image on `imageText` slides; stripped when layout ≠ `imageText`
- **`contentToFormFields`**, **`formFieldsToContent`** — plan/edit form binding
- **`buildSlideContentWithImage`** — `src/server/slide-image.ts` (upload / future AI persist helper)
- **`comparisonBulletsBlocksToText`**, **`comparisonTextToBulletsBlocks`**

### `src/lib/openai-chat-model.ts`

- **`OpenAiChatModel`**, **`openAiChatModelSchema`**, **`OPENAI_CHAT_MODEL_DEFAULT`**
- **`readStoredOpenAiChatModel`**, **`persistOpenAiChatModel`** — client localStorage

### `src/lib/openai.ts`

- **`openai`** — OpenAI client (`server-only` consumers)

### `src/server/ai/deck-plan-openai.ts`

- **`generateDeckPlanViaOpenAi`** — structured deck plan from prompt + `DeckSettings` + brand kit
- **`rewriteSlideViaOpenAi`** — single-slide rewrite with `focus` (`title` \| `body` \| `bullets` \| `image` \| `notes` \| `all`)

### `src/server/queries/`

| Module | Exports |
|--------|---------|
| `deck-create.ts` | `resolveBrandKitForNewDeck` |
| `deck-dashboard.ts` | `getDeckTabCounts`, `listDecksForDashboard` |
| `brand-kit.ts` | `getBrandKitDetail`, `listBrandKitsWithDeckCounts`, defaults |

### Other `src/lib` (UI helpers)

- **`slide-viewport.ts`** — `SLIDE_LOGICAL_W`, `SLIDE_LOGICAL_H` (1280×720)
- **`color-contrast.ts`** — `contrastRatio`, `passesAaNormalBody` (brand kit compliance)
- **`format-relative-updated-at.ts`** — dashboard relative timestamps
